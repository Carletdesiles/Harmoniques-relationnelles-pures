# # Harmoniques relationnelles pures

Petite modélisation ontologique : vibration née uniquement des distances R_ij entre points.

Pas de temps, pas d'onde externe, pas de champ ajouté.

Inspiré :
- Julian Barbour (formes relationnelles, complexité sans espace-temps absolu)
- Stéphane Dugoffson (espaces connectifs comme résidu d'interactions entre espèces)
- Benoît Mandelbrot (rugosité fractale à toutes les échelles)

Le réseau vibre quand deux points se rapprochent : 1/R monte → phase oscille → produit global explose = résonance.

Exécute le script ou le notebook pour voir.
